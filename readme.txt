== Description ==

<p><em>En fran&ccedil;ais</em></p>
<p><strong>Tag-to-Link (jQueryRank sculpting)</strong> fonctionne en anglais et en fran&ccedil;ais<br/>
<strong>Tag-to-Link</strong> permet de modifier des balises HTML portant une classe et un attribut sp&eacute;cifique en lien hypertexte afin de mieux g&eacute;rer le jus des liens (PageRank Sculpting). Moins vous avez de liens dans une page, plus chaque lien profite du jus, Tag-to-Link est l&agrave; pour &ccedil;a...</p>

<p><em>For English people</em></p>
<p><strong>Tag-to-Link (jQueryRank sculpting)</strong> works in English and in French<br/>
<strong>Tag-to-Link</strong> used to modify HTML tags with a class and a specific attribute in hyperlink to better manage the link juice (PageRank Sculpting). Unless you have links in a page, each benefiting more link juice, Tag-to-Link is here for ...</p>

== Installation ==

<p><em>En fran&ccedil;ais</em></p>
<p>
1. Recopier le contenu de l'archive dans le r&eacute;pertoire des extensions (wp-content/plugins) et activer dans le panneau des extensions du backoffice de Wordpress.<br/>
2. Choisir une classe de s&eacute;lecteur ('linktoggle' par d&eactute;faut), un attribut devant contenir l'URL compl&egrave;te du lien ('title' par d&eactute;faut) et l'action de la souris qui provoque la transformation (le lien doit contenir "http://" ou "mailto:" etc.)<br/>
3. Ecrire des contenus et passer en HTML pour ajouter le s&eacute;lecteur (id ou class) et l'attribut choisi dans les balises qui doivent &ecirc;tre modifi&eacute;es en lien lors d'un survol (voir screenshot).<br/>
N.B. : donner un m&ecirc;me style &agrave; la balise initiale et au lien final pour ne pas tromper les internautes...
</p>

<p><em>For English people</em></p>
<p>
1. Copy the content of the archive in "wp-content/plugins" and activate the plugin.<br/>
2. Choose a selector (default : linktoggle) and an attribute to contain the URL link (default : title) before processing.<br/>
3. Write content and switch to HTML to add the selector (id or class) and the attribute in the tags that must be changed in links (see screenshot).<br/>
Note: give the same style to the initial tag and final link not to mislead users ...
</p>
